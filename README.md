## 📝 주요 기능
### 🙌🏻 공통 서비스
서비스 전반의 편의성과 사용자 인증을 담당
- **사용자 인증 및 관리:** 회원가입, 로그인/로그아웃 및 내 정보 관리 기능을 통한 보안 유지
- **반응형 공통 레이아웃:** 서비스 전반에 걸쳐 일관된 UI를 제공하는 GNB, Footer, Sidebar 레이아웃
- **알림 시스템:** 실시간 알림 기능을 통해 예약 상태 변경 및 피드백을 사용자에게 즉시 전달

### 👀 체험 사용자 (Guest)
일반 사용자가 체험을 즐기고 기록하는 영역
- **상품 탐색 및 상세 확인:** 카테고리 필터링과 검색을 통해 원하는 체험을 찾고, 상세한 소개와 위치(지도) 확인
- **예약 프로세스:** 원하는 스케줄을 선택하여 체험을 예약하고, 내 예약 내역을 상태별(신청/취소/완료)로 확인
- **리뷰 CRUD:** 체험 완료 후 소중한 경험을 기록하고, 내가 쓴 리뷰를 조회 및 수정/삭제하는 기능

### ✏️ 체험 등록자 (Host)
데이터 생성 및 관리 영역
- **체험 등록 및 수정:** 상세 정보, 카테고리, 가격, 주소 등을 입력하여 체험 상품을 생성하고 관리하는 기능
- **이미지 업로드 시스템:** 배너 이미지 및 다중 소개 이미지를 서버로 업로드하고 미리보기 기능 제공
- **스케줄 및 예약 관리:** 체험 가능한 날짜와 시간을 설정하고, 들어온 예약 신청을 수락하거나 거절하는 기능
- **데이터 정합성 유지:** 등록/수정 시 입력 폼의 유효성을 검사하고, 안정적인 API 통신으로 데이터 무결성 확보
